Font System Demo

This page demonstrates how BananaPrep's centralized font system works. You can use the selector below to change fonts across the entire application.

How it works

The font system works by using CSS variables defined in a single file (font-variables.css). All other CSS files reference these variables, ensuring consistency across the app.

Implementation

To implement this system, we created:

  1. font-variables.css - Defines all font-related CSS variables
  2. Updated font.css - Uses these variables for typography
  3. font-switcher.js - JavaScript to change fonts dynamically

How to Use in Your HTML

Simply include the CSS files in the correct order:

<link rel="stylesheet" href="css/font-variables.css"> <link rel="stylesheet" href="css/styles.css"> <link rel="stylesheet" href="css/font.css">

To add the font switcher to any page, include the font-switcher.js script and add a container with the ID "fontSwitcherContainer":

<div id="fontSwitcherContainer"></div> <script src="js/font-switcher.js"></script>

To change fonts programmatically from any JavaScript code:

// Change to a different font window.BananaPrepFonts.changeFont('serif'); // Available options: 'default', 'system', 'serif', 'openDyslexic'
